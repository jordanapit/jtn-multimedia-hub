# JTN Multimedia Hub
JTN Multimedia Hub adalah aplikasi pusat pelayanan Seksi Multimedia HKBP Jatinegara yang memilki banyak fitur, seperti presensi, unduh templat, unggah bahan pelayanan, unduh bahan pelayanan di Gereja, rekap presensi, rekap transport (Persembahan Kasih), dan lain-lain.

Catatan:
Git ini merupakan Git untuk Aplikasi JTN Multimedia Hub versi Windows. Untuk web utama JTN Multimedia Hub tidak dipublikasikan.
